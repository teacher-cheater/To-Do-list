# To-Do-list

Приложение для выполнения поставленных на день/неделю/месяц задач. Все выполненные задачи можно отметить, а позже их удалить либо по одной, либо все сразу. Также у приложения есть регистрация, чтобы было понятно, чьи задачи выполняются. Если вы передумали или список задач поменялся, то можно список дел очистить.
Приложение также доступно на мобильных устройствах, удобно брать с собой в магазин, чтобы отмечать выбранные продукты) 

[Попробовать онлайн](https://teacher-cheater.github.io/To-Do-list/)
